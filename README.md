# 🌍 Translify AI

A modern AI-powered language translation web application built using **Python, Flask, HTML, CSS, and JavaScript**.

Translify AI allows users to translate text between multiple languages with automatic language detection, voice input, text-to-speech, and persistent translation history.

---

## ✨ Features

* 🌐 Multi-language Translation
* 🔍 Automatic Language Detection
* 🎤 Voice Input
* 🔊 Text-to-Speech
* 📜 Persistent Translation History
* 📂 Collapsible History Panel
* 🌓 Dark / Light Theme
* 📋 Copy Translation
* 🗑 Clear Translation
* 📱 Modern Responsive User Interface

---

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Translify-AI.git
```

### 2. Open the project folder

```bash
cd Translify-AI
```

### 3. Install dependencies

```bash
pip install flask
pip install deep-translator
pip install langdetect
```

### 4. Run the application

```bash
python app.py
```

### 5. Open in browser

```
http://127.0.0.1:5000
```
